Services

By Our Application

WebLegion provides all the services that a user will need in order to decide the service provider he/she wanst to use. The main Service includes fare comparison side by side of Uber and Lyft for nay route that user wants to travel. The user is also provided with a highchart showing the comparison in fare of both the services, making it more comfortable for user to decided. Along with this, the user is also provided with the route in the map and traffic in that route that user might get. These services will help user in make decesion making easier.

fare Comparison

We care about our passenger's budget and preferences

The fare comparison includes fare from Uber and Lyft side by side, which gives user option according to his/her budget, comfort and preferences. We have acquired these fare estimates by the Uber and Lyft API mkaing them the real time data a passenger may get. This makes it easier for passenger to be hassle free by not hopping from one application to another and book the cab by directly redirecting to the original application without filling up the pick up and drop addresses.

HighCharts comparison

For QUICK and SIMPLE look

The Highcharts services for live data is a booming service used by major tech and corporate giants in the world. We have used this service to fetch real time data from the Uber and Lyft to make it easier and simpler for user to have a quick view of all the option a user might get. Highcharts are interactive JavaScript charts, offering an easy way of adding interactive charts to a web site or web application

Live Traffic

We care about passangers comfort

WebLegion also provides the live traffic situation in the routes so user can decide taking a cab according to his/her comfort. A user may not need to stop the application and serach seperately for the traffic updates and then get back to the application to book. These traffic updates are real time update from Google API displaying the traffic situation in perticular route in a traditional google style by showing different colors for light, medium and heavy traffic so that user do not get confused.
